>  How I did it:

So I didn't atually make these levels. These are from codingbat, a website with TONS of practice problems in Java and Python. I used the urllib module to extract all levels, solutions, and their test code to test my user's code.

Usage: In the parameters of Main(), enter the start_on (a number that is 1 greater than the length of the dictionary in resources/levels.txt) and a category (String-1, Warmup-2, etc.)
It will give you key-values that you can put into the dictionary resources/levels.txt. Then, you have more levels! >  Return home.
